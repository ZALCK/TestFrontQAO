# QUADRA AO - Test - Art-Institut-Of-Chicago

## Fonctionnalités implementée
- Parcourir la gallery des oeuvres d'art
- Recherche les oeuvres d'art

```sh
npm install
```

npm run dev

### Compile and Minify for Production
